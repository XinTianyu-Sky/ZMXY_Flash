# 造梦西游3再续天庭 — 版本变更记录

基于 B 站"机智的远远"社区的二次开发单机版。作者：夕眼、石竹。

---

## v0.85 (2023-01-22)

- 昆仑山每10波获取到积分后自动存档，地狱模式双倍积分
- 取消困难模式秒杀，bingo 改为扣除50%血量、10%魔法值
- 存档间隔改为10秒
- 减少技能蓝耗
- 刑天压制效果减少到25级
- 增加游戏设置：默认关闭声音、关闭技能音效
- 悟空、八戒被动更改：
  - 嗜血暴击：3 + level/2 %
  - 刃甲防御：level × 5
  - 刃甲回血概率：10 + level/2 %，回血值为攻击的80%（受渊邪、流邪加成）
- 降低前期难度
- 修复太极八卦属性重置 bug
- 修复属性重置传承法器 bug
- 优化四神兽技能冷却机制
- 修复挑战心魔5的词条错误

## v0.82 (2023-01-14)

- 修复八戒冻结 bug
- 修复沙僧残影 bug
- 优化雪山降雪范围
- 等级上限提升到80级
- 新增上清、玉清、太清印记

## v0.7

**大改版，更新内容极多：**

- 重置所有BOSS血量，取消技能连招机制，改为每隔一段时间随机出技能
- 重置所有人物技能数值及部分技能机制，主被动技能升级受等级限制（上限50级）
- 超等级被动会被强制降到最大等级，灵魂返还；主动技能超等级的提供孟婆药剂补偿
- 新增时光之舟（40级）：七叶树窟、慕山；罗睺掉落烛时金轮
- 御马监彩蛋BOSS → 白骨秘境 → 白骨精掉落饰品
- 蟠桃园猕猴王（40级后）→ 六耳猕猴 → 烛时金轮2-5级升级道具
- 重置回血回蓝属性以平衡技能蓝耗
- 宠物与法宝伤害调整
- 汪汪挑战条件更改
- 所有BOSS爆率再度提升
- 紫金铃铛改为提供无敌+回复蓝血
- 困难模式微调：BOSS血量和防御更高，掉率/灵魂/经验更高
- 玲珑宝塔出怪数量每层一致
- 修复：法宝界面掉血蓝 bug、宠物玄武青龙释放 bug、捕捉宠物灵魂负数 bug
- 新增彩蛋
- 技能改动：烈焰闪加0.125s前摇、火魔斩最后一斩落地、唐僧水幻影升级提升效果和移速、八戒土魔刺无双改10根、沙僧巫毒娃娃血量可升级
- **被动技能不要升级！**

## v0.1 (2022-09)

- 东天王殿小怪替换为邪沙僧（2%概率掉1级强化石），BGM替换

- 大部分强化装备属性为原版2倍
- 背景音乐全部替换为高音质版

- 补偿礼包"确定"/"取消"内容不同
- 怪物掉落物品不会消失
